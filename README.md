# Shopify Cinematic Scroll Hero

A production-ready Shopify Hero Banner featuring cinematic GSAP scroll animations, responsive design, Theme Editor customization, accessibility, and Core Web Vitals optimization.

## ✨ Features

- GSAP ScrollTrigger animations
- Shopify Theme Editor support
- Responsive design
- Core Web Vitals optimized
- SEO friendly
- Accessibility focused
- Horizon Theme compatible
- Shopify Online Store 2.0 compatible
