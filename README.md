# Cinematic Scroll Hero for Shopify

A premium, production-ready cinematic scroll animation hero section for Shopify themes. Built with GSAP, ScrollTrigger, and modern web standards.

![Cinematic Scroll Hero](screenshots/hero-preview.gif)

## ✨ Features

- 🎬 **Cinematic scroll animation** — Images animate and spread as users scroll
- 🎯 **GSAP + ScrollTrigger** — Smooth, performant animations
- 📱 **Fully responsive** — Looks great on all devices
- ♿ **Accessible** — WCAG 2.2 AA compliant
- 🚀 **Performance optimized** — Lighthouse 95+ ready
- 🎨 **Theme Editor controls** — Extensive customization options
- 🧩 **Modular architecture** — Clean separation of concerns
- 🔄 **Dynamic sections** — Full Shopify Online Store 2.0 support
- 🌐 **SEO friendly** — Semantic HTML, proper heading hierarchy
- 📦 **Zero dependencies** — Only requires GSAP (included via CDN or theme)

## 📋 Requirements

- Shopify Online Store 2.0 theme
- GSAP 3.x (included via theme or CDN)
- Modern browser support (Chrome, Firefox, Safari, Edge)

## 🚀 Quick Start

### 1. Add GSAP to your theme

Add GSAP to your theme.liquid before `</head>`:

```liquid
<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/gsap.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/ScrollTrigger.min.js"></script>
