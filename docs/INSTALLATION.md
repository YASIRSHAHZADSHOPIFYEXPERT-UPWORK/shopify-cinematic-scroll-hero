# Installation Guide

## Requirements

- Shopify Online Store 2.0
- Horizon Theme (Recommended)
- Dawn Theme (Supported)

## Installation

1. Copy the Liquid section into:
   sections/cinematic-scroll-hero.liquid

2. Copy the CSS file into:
   assets/cinematic-scroll-hero.css

3. Copy the JavaScript file into:
   assets/cinematic-scroll-hero.js

4. Include the CSS and JavaScript files within the section.

5. Open Shopify Theme Editor.

6. Add the **Cinematic Scroll Hero** section to your desired template.

7. Configure the section settings.

8. Save the theme.

## Compatibility

- Horizon
- Dawn
- Shopify Online Store 2.0

## Performance

- Optimized for Core Web Vitals
- Lazy loading
- Responsive images
- Mobile optimized
